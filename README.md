# Jenkins CI/CD Pipeline Examples

This repository contains examples of using Jenkins for continuous integration (CI) and automated builds.

The projects demonstrate how Jenkins pipelines can build, test, and manage Java applications.

--------------------------------------------------

PROJECTS

BankAccount-Java
A Java project built using Maven with automated Jenkins pipeline execution.

BankAccount
A small example project used to demonstrate Jenkins build and testing steps.

Orders-Service
Example service project configured with Jenkins pipeline automation.

--------------------------------------------------

JENKINS PIPELINE

The repository includes Jenkinsfiles that define the CI pipeline.

Typical pipeline steps include:

- Pull source code from GitHub
- Compile the project
- Run tests
- Build the application
- Report build results

--------------------------------------------------

TECHNOLOGIES USED

- Jenkins
- Java
- Maven
- Git
- CI/CD Pipelines

--------------------------------------------------

GOAL

The goal of this repository is to practice building automated CI/CD pipelines using Jenkins.
